# Sales Perfomance and analytics project

## Project Overview
The project analyzes sales performance across region, countries, channel and products.

### Objectives
Retail Businesses need to understand historical sales perfomance in order to improve strategic planning, and anticipate future demands.

The project aims to answer the followin business questions:
- Which Region generates the highest profit?
- Which channel generates the highest profit?
- What are the sales trend over time?
- Which product category generated highest revenue?
- Service leve perfomance across the regions

# Project Workflow
The project follows a structured analytics workflow:

### Data Preparation
- Data validation and structure review
- Data type verification
- Missing values checks

### Exploratory data analysis
- Cohort analysis
- Sales by Region and salesperson
- Revenue by salesperson
- 7-day sales forecast

### Dashboard Development
- Sales performance overview
- Sales trend analysis
- Interactive filtering and visualization

## Key Insights
Some of the key insight from the analysis include:
- Asia generated hihest revenue among all the regions.
- Revenue trends shows that July 2024 was the strongest sales month, generating highest monthly revenue.
- Service level performance (orders delivered within the 7-day target) was strongest in Africa, with Asia having the lowest.
- Networking products generate the highest revenue among all the product categories.
